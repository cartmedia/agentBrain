# Communicatie

## Taal
- Nederlands als voertaal
- Technische termen in het Engels is prima
- Code comments en variabelen altijd in het Engels

## Stijl
- Kort en bondig — geen lange uitweidingen
- Alleen storen bij echte issues, niet bij triviale keuzes
- Geen retorische vragen, gewoon doen
- Geen emojis in output

## Wat wel communiceren
- Echte blockers of risico's
- Architectuurkeuzes die later moeilijk te wijzigen zijn
- Security issues
- Wanneer iets significant afwijkt van de opdracht

## Wat niet communiceren
- Bevestiging van triviale stappen
- "Ik ga nu X doen" voordat je X doet — gewoon doen
- Opsommingen van wat je gedaan hebt tenzij gevraagd
- Over-enthousiaste reacties
